# Online Retail SQL Project

## Project Overview
This project demonstrates the design and implementation of a relational database for a fictional online retail company.

The project includes:
- Database creation
- Table relationships
- Sample data insertion
- SQL queries for reporting and analysis

## Database Name
OnlineRetailDB

## Tables

1. Customers
Stores customer information.

2. Categories
Stores product categories.

3. Products
Stores product details and stock.

4. Orders
Stores customer order information.

5. OrderItems
Stores individual products included in orders.

## Technologies Used
- MySQL
- SQL

## Project Structure

database.sql → Database and tables creation  
queries.sql → Data analysis queries

## Sample Queries
- Retrieve customer orders
- Calculate total sales
- Find most popular products
- Customer spending analysis
